# display
```js
// display: none; 会隐藏元素，并且释放原有空间

// display: inline; 行元素
1. 多个元素占一行
2. 不可以设置宽高
行元素：span a strong

// display: block;  块元素
1. 自己占一行
2. 可以设置宽高
块元素：div h1-h6 ul li

// display: inline-block    行内块元素
1. 多个元素占一行
2. 可以设置宽高
行内块元素：img button input
```

# visibility
```js
// display: none;
会隐藏元素，并且释放原有空间

// visibility: hidden;
隐藏元素，并且原有空间不释放

// visibility: visible
元素可见
```
