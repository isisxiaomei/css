# 边框
# 1. border-radius
+ `border-radius`: 设置圆角边框

```css
/* 示例1： */
border-radius: 10px;
border-radius: 10px 40px;只有两个值表示 左上角和右下角 右上角和左下角 （对角线）

/* 示例2： */
.box {
  border: 10px solid rebeccapurple;
  border-radius: 1em;
  border-top-right-radius: 10% 30%;
}
```

# 2. border-image
+ `border-image`: 设置边框图片
