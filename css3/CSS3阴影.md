# 1 盒子阴影
+ `box-shadow`: 用于在元素的框架上添加阴影效果
```css
/* 示例1： */
/* x偏移量：阴影在水平方向的偏移量（正数向右 负数向左）
 | y偏移量 ：阴影在垂直方向的偏移量（正数向下 负数向上）
 | 阴影模糊度半径： 代表阴影模糊度，不能为负数
 | 阴影颜色 */
box-shadow: 10px 5px 5px black;

/* 示例2：设置多个阴影用逗号隔开 */
box-shadow: 10px 5px 5px red,
            20px 10px 5px blue;
```
# 2 文字阴影
+ `text-shadow`：设置文字阴影；(使用和盒子阴影相同)
```css
/* 示例1： */
box-shadow: 10px 5px 5px black;
```
